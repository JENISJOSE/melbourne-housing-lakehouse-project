## Melbourne Housing Lakehouse Analytics Platform
This project demonstrates an end-to-end Data Engineering pipeline built using Databricks, PySpark, Delta Lake, and SQL. The solution implements Medallion Architecture (Bronze, Silver, and Gold layers) to ingest, clean, transform, and analyze Melbourne housing market data.

The pipeline processes raw property transaction data, applies data quality checks and transformations, and generates business-ready datasets for analytics and reporting. Gold-layer datasets provide insights into average property prices, suburb-level market performance, and property type trends.

## Technology Stack

* Databricks
* PySpark
* Delta Lake
* Spark SQL
* GitHub
* Power BI (planned)

## Architecture

Raw CSV Data → Bronze Layer → Silver Layer → Gold Layer → Analytics & Reporting

## Key Business Metrics

* Average Property Price by Suburb
* Property Count by Suburb
* Average Property Price by Property Type
* Melbourne Housing Market Trends
